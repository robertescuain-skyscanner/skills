# Claude Code Skills

A collection of custom skills for [Claude Code](https://claude.com/claude-code).

## About

This repository contains reusable skills that extend Claude Code's capabilities. Skills can be shared and imported into your local Claude Code environment.

## Available Skills

- **tdd**: Test-driven development workflow skill

## Usage

To use a skill from this repository, copy the skill directory to your local Claude Code skills directory (typically `~/.claude/skills/`).

## Structure

Each skill directory contains:
- `skill.json`: Skill metadata and configuration
- Prompt file(s): The skill's instructions and behavior
